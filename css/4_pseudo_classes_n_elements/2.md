## 2. Falta

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Job Application Form</title>
    <link href="styles.css" rel="stylesheet">

</head>

<body>
    <div class="container">
        <form>
            <label id="nombre">Full name:</label>
            <input type="text" id="name">
            <label id="email">Contact email:</label>
            <input type="email" id="email">
            <label id="pos">Position:</label>
            <select id="position">
                <option>CEO</option>
                <option>Manager</option>
                <option>Senior</option>
                <option>Junior</option>
            </select>

            <fieldset class="radio-group">
                <input id="full-option" type="radio" name="availability" />
                <label for="full-option">Full-time</label>
                <input id="part-option" type="radio" name="availability" />
                <label for="part-option">Part-time</label>
            </fieldset>

            <textarea id="message"></textarea>

            <button type="submit">Submit</button>
        </form>
    </div>
</body>

</html>
```

```css
.container {
  width: 80vw;
  margin: 10px auto;
  padding: 40px;
  box-shadow: 1px 4px grey;
}

#nombre, #email, #pos {
  font-size: 16px;
  font-weight: bold;
  display: block;
  margin: 10px 0px;
}

form > input, select {
  width: 100%;
  margin-bottom: 1px;
}

input:focus, textarea:focus {
  border: 1px solid yellow;
}

input:invalid, select:invalid, textarea:invalid {
  border-color: red;
}

input:valid, select:valid, textarea:valid {
  border-color: green;
}

button:hover {
  background-color: blue;
}

.radio-group input[type="radio"]:checked {
  border-color: green;
  background-color: lightgreen;
  box-shadow: 1px 4px grey;
}

input[type="radio"]:checked + label {
  color: green;
}

input:first-of-type {
  border-radius: 10px;
}
```
